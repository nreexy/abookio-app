<p>Track the latest updates and improvements to Abookio.</p>

<h2>2025-11-25 Version 1.1</h2>

<h3>Audiobookshelf</h3>
<ul>
  <li><strong>Fixed:</strong> Resolved an issue where folder-based audiobooks were missing from the import list.</li>
  <li><strong>Fixed:</strong> Corrected size and duration calculations in the detail view; multi-file audiobooks now accurately sum individual file values.</li>
  <li><strong>New:</strong> Audiobookshelf can now be selected as a metadata provider in Settings/Metadata/Metadata Providers.</li>
</ul>

<h3>Jellyfin</h3>
<ul>
  <li><strong>Fixed:</strong> Added reliable support for importing multi-file MP3 audiobooks.</li>
  <li><strong>Fixed:</strong> Corrected size and duration calculations in the detail view.</li>
</ul>

<h3>General Improvements & Fixes</h3>
<ul>
  <li><strong>Plex:</strong> Resolved an issue preventing the import of audiobooks with spaces in their filenames.</li>
  <li><strong>OneDrive:</strong> Fixed an issue where accounts were not persisting after login.</li>
  <li><strong>WebDAV:</strong> Resolved a bug that incorrectly displayed authentication errors for valid accounts.</li>
  <li><strong>Library:</strong> Fixed a UI bug where the active "Sort By" selection was not updating correctly.</li>
  <li><strong>Website:</strong> Added Blog and Wiki sections.</li>
</ul>
